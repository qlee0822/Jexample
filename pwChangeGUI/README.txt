1. pwtype.txt 작성 (필수는 아님)
 - 정해진 양식에 맞게 작성
 - 반드시 사이는 TAB키로 띄워야 함

2. pwChangeGUI.exe 실행

3. 기존 패스워드 파일 열기
 - 참고하고자하는 시트 선택
   ※ 엑셀은 암호가 해제되어 있어야 함

4. 패스볼트 파일 열기
 - 참고하고자 하는 시트 선택
   ※ 엑셀은 암호가 해제되어 있어야 함
 - 만일 패스볼트 파일이 아직 생성 전 이라면 생략 가능

5. 패스워드 규칙 설정 입력(※ 하나라도 입력이 안되어 있으면 실행이 안됨)
 (1) pwtype.txt가 정상적으로 입력되어 있으면 자동으로 Type별 문자를 불러옴
 (2) 파일 - 패스워드타입 열기 메뉴에서 정해진 양식의 txt파일로 불러 들일 수 있음
 (3) 직접 입력 가능

6. 패스워드 변경 스크립트 생성 여부 체크
 - 체크시 : K열에 패스워드 변경 스크립트 생성(리눅스 서버만 사용가능)

7. 패스워드 변경 버튼 클릭

8. PassVault 변경 버튼 클릭
   ※ 순서중요: 패스워드 변경 버튼 클릭 다음에 PassVault 변경 버튼 클릭

9. 저장 버튼 클릭
 - 저장된 엑셀에서 ChangPW_Sheet에 변경된 패스워드 확인
 
10. 닫기
